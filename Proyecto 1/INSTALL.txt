Para instalar y ejecutar correctamente el programa debe seguir los siguientes pasos:
    1. Asegurarse que el archivo "guardians_tournament_data.txt" se encuentre en la misma
        carpeta que el archivo "proyecto_unidad1.c".
    2. Abrir el archivo "proyecto_unidad1.c" con un compilador para lenguaje C.
    3. Ejecutar el archivo "proyecto_unidad1.exe".
